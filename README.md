# kinetic-gain-directory

Curated directory of domains known to publish [Kinetic Gain Protocol Suite](https://suite.kineticgain.com) documents at canonical `/.well-known/` URLs.

Live at **[directory.kineticgain.com](https://directory.kineticgain.com)**.

## Submit a domain

1. Publish at least one Kinetic Gain document at a canonical well-known path.
2. Verify with [walker.kineticgain.com](https://walker.kineticgain.com) — at least one document must come back `FOUND`.
3. Open a PR adding your entry to this repo.
